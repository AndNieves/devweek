# ParkingEndava

## Requisites
+ node
+ angular cli
+ firebase

## Environment set up

### Firebase
1) Install Firebase tools:
  * **npm install -g firebase-tools**
2) Login to firebase:
  * **firebase login**
3) Initialize firebase app:
  * **firebase init**
4) Configure deploy folder in firebase config file
  * Typically **/dist**
5) To deploy, use **firebase deploy**
